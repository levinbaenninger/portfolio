A modern, performant portfolio website built with industry best practices and comprehensive quality assurance.

## ✨ Features

- **🚀 Next.js 15** with App Router and Turbopack for lightning-fast development
- **📱 Responsive Design** with Tailwind CSS 4.0
- **⚡ Performance Optimized** with automatic optimization and monitoring
- **♿ Accessibility First** with WCAG 2.1 AA compliance testing
- **🔒 Security Focused** with automated vulnerability scanning
- **📊 Quality Assurance** with comprehensive CI/CD pipeline
- **🎨 Type Safety** with TypeScript and strict linting via Ultracite

## 🛠️ Tech Stack

### Core Technologies

- **Framework**: Next.js 15 with App Router
- **Language**: TypeScript
- **Styling**: Tailwind CSS

### Development & Quality Tools

- **Linting & Formatting**: Ultracite (Biome-powered)
- **Git Hooks**: Husky with lint-staged
- **Bundle Analysis**: Next.js Bundle Analyzer
- **Package Manager**: pnpm with frozen lockfiles

### Testing & Monitoring

- **Performance**: Lighthouse CI with strict thresholds
- **Accessibility**: pa11y with WCAG 2.1 AA testing
- **Security**: CodeQL analysis + npm audit
- **Dependencies**: Automated updates via Dependabot
